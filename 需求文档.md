在线建表系统（前端版）需求文档
适用场景：数据库表可视化配置、动态表单自动生成平台
技术栈：Vue3 + TypeScript + Element Plus + Qiankun（微前端）
数据方案：Mock.js 模拟接口 + localStorage 持久化存储
核心范围：仅实现表配置模块（数据录入 / 预览 / 数据源管理暂不实现）
一、文档概述
1.1 项目背景
基于 Vue3 技术栈开发在线可视化建表工具，用户可通过拖拽、动态表单配置完成表结构定义，系统自动根据表配置生成对应表单。
本项目仅实现表配置核心功能，数据录入、数据源管理、后端接口均不实现。
1.2 功能范围
表配置列表页（查看 / 编辑 / 删除）
表基础信息配置
表字段动态配置（核心）
表索引动态配置
整体配置保存 / 重置 / JSON 预览
1.3 技术栈
框架：Vue3 + <script setup>
语言：TypeScript
UI 组件：Element Plus
微前端：Qiankun（集成适配）
模拟数据：Mock.js + localStorage
拖拽：sortable.js（推荐）

三、功能需求明细
3.1 表配置列表页
页面功能：展示所有已创建的表配置
表格列：表名称、表编码、表描述、表类型、创建时间、更新时间
操作按钮：
新建表配置（跳转编辑页）
查看（只读展示）
编辑（进入编辑页）
删除（二次确认）
支持表编码模糊搜索
数据来源：Mock + localStorage
3.2 表基础信息配置（标签页 1）
3.2.1 配置项
表格
配置项	控件	必填	规则
表名称	输入框	是	不能为空
表编码	输入框	是	不能为空 + 全局唯一（Mock 校验）
表描述	文本域	否	-
表类型	下拉单选	是	单表 / 主表 / 附表
关联分组	自定义分组	-	新增 / 编辑 / 删除 / 拖拽排序
主表关联	下拉单选	附表时必填	仅附表显示，选择已配置的主表
3.2.2 表单分组功能
新增分组
编辑分组名称
删除分组
拖拽调整分组顺序
字段可归属到某一分组
3.2.3 校验规则
表名称、表编码必填
表编码不能重复
附表必须选择主表
分组名称不可重复
3.3 表字段列表配置（核心动态表单）
功能：动态增删改查、拖拽排序、批量操作
3.3.1 字段配置项（每一行）
① 基础属性
字段名称（输入框，必填）
字段编码（输入框，必填 + 唯一）
字段类型（下拉：varchar/int/date/text/datetime 等）
字段长度（输入框）
默认值（输入框）
是否必填（开关）
是否主键（单选互斥，仅允许一个）
② 展示属性
表单组件类型（下拉）
输入框
数字框
单选
多选
下拉
日期
时间
富文本
字段说明（输入框）
排序号（数字框，自动生成，可改）
③ 归属属性
所属分组（下拉：关联基础信息中配置的分组）
3.3.2 操作功能
新增字段行
编辑字段行
删除单行
拖拽调整字段顺序
批量删除选中字段
一键清空所有字段
3.3.3 校验规则
字段名称、字段编码必填
字段编码不可重复
主键全局唯一（仅允许一个字段为主键）
所属分组必须选择已存在的分组
3.4 表索引列表配置
功能：动态增删改查、拖拽排序
3.4.1 索引配置项
索引名称（输入框，必填）
索引类型（下拉）
普通索引
唯一索引
主键索引
关联字段（多选框，来源上方字段列表）
索引说明（文本域）
3.4.2 操作功能
新增索引行
删除单行
拖拽调整索引顺序
3.4.3 校验规则
索引名称必填
唯一索引 / 主键索引的关联字段不可重复
主键索引只能存在一个（与字段主键互斥可扩展）
3.5 表配置全局操作
3.5.1 保存配置
统一校验：表信息、字段、索引全部校验
校验通过 → 保存到 localStorage（Mock 存储）
保存成功提示 → 返回列表
3.5.2 重置配置
一键清空：
表基础信息
所有字段
所有索引
所有分组
清空前二次确认
3.5.3 预览配置
点击弹出弹窗
展示完整表结构 JSON（格式化高亮）
仅预览，不提交、不处理
四、TypeScript 类型定义（核心）
typescript
// 表类型
export enum TableType {
  SINGLE = 'single',    // 单表
  MAIN = 'main',        // 主表
  SLAVE = 'slave'       // 附表
}

// 字段类型
export enum FieldType {
  VARCHAR = 'varchar',
  INT = 'int',
  TEXT = 'text',
  DATE = 'date',
  DATETIME = 'datetime',
  TIME = 'time'
}

// 表单组件类型
export enum FormComponentType {
  INPUT = 'input',
  NUMBER = 'number',
  RADIO = 'radio',
  CHECKBOX = 'checkbox',
  SELECT = 'select',
  DATE = 'date',
  TIME = 'time',
  RICH_TEXT = 'richText'
}

// 索引类型
export enum IndexType {
  NORMAL = 'normal',
  UNIQUE = 'unique',
  PRIMARY = 'primary'
}

// 分组
export interface TableGroup {
  id: string;
  name: string;       // 分组名称
  sort: number;        // 排序
}

// 字段
export interface TableField {
  id: string;
  fieldName: string;   // 字段名称
  fieldCode: string;   // 字段编码
  fieldType: FieldType;
  length?: number;
  defaultValue?: string;
  required: boolean;
  isPrimary: boolean;
  componentType: FormComponentType;
  fieldDesc?: string;
  groupId: string;     // 所属分组ID
  sort: number;
}

// 索引
export interface TableIndex {
  id: string;
  indexName: string;
  indexType: IndexType;
  fieldIds: string[];  // 关联字段ID列表
  indexDesc?: string;
  sort: number;
}

// 表基础信息
export interface TableInfo {
  tableName: string;
  tableCode: string;
  tableDesc?: string;
  tableType: TableType;
  mainTableCode?: string; // 附表关联主表编码
}

// 表完整配置
export interface TableConfig {
  id: string;
  info: TableInfo;
  groups: TableGroup[];
  fields: TableField[];
  indexes: TableIndex[];
  createTime: string;
  updateTime: string;
}
